Request Flow: User Question → Audit
1
User sends chat message
React app captures user input with session context (org_id, user_id, chatbot_id)
2
POST request to Quarkus Gateway
React → POST /api/chat with { prompt, sessionId, organizationId, chatbotId, model }
3
Proxy forwards to Third-Party API
Quarkus adds API keys, forwards request to OpenAI/Claude/etc., establishes SSE stream
4
Stream response back to React
Quarkus pipes streaming response chunks back to frontend in real-time
5
Publish audit event (async)
Quarkus publishes complete conversation to Kafka/RabbitMQ (non-blocking)
6
Audit Service consumes event
Separate Quarkus consumer picks up audit event from message queue
7
Upload conversation to Azure Blob
Full prompt + response saved as JSON blob (e.g., chats/2024/01/29/session-abc-123.json)
8
Store metadata in PostgreSQL
Insert record with conversation_id, session_id, org_id, blob_reference, timestamps, token counts
PostgreSQL Schema: chat_audits Table
CREATE TABLE chat_audits (
    id                  UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    conversation_id     UUID NOT NULL,
    session_id          VARCHAR(255) NOT NULL,
    question_id         VARCHAR(255),
    organization_id     VARCHAR(255) NOT NULL,
    user_id             VARCHAR(255),
    chatbot_id          VARCHAR(255) NOT NULL,
    model               VARCHAR(100) NOT NULL,
    
    -- Timestamps
    created_at          TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    completed_at        TIMESTAMPTZ,
    
    -- Metrics
    token_count_prompt  INTEGER,
    token_count_response INTEGER,
    latency_ms          INTEGER,
    
    -- Storage reference
    blob_reference      TEXT NOT NULL,
    blob_container      VARCHAR(255) DEFAULT 'chat-audits',
    
    -- Additional metadata
    metadata            JSONB,
    status              VARCHAR(50) DEFAULT 'completed',
    error_message       TEXT,
    
    -- Indexes
    CONSTRAINT fk_organization FOREIGN KEY (organization_id) 
        REFERENCES organizations(id)
);

CREATE INDEX idx_session_timestamp ON chat_audits(session_id, created_at DESC);
CREATE INDEX idx_org_timestamp ON chat_audits(organization_id, created_at DESC);
CREATE INDEX idx_user_timestamp ON chat_audits(user_id, created_at DESC);
CREATE INDEX idx_chatbot_model ON chat_audits(chatbot_id, model);
CREATE INDEX idx_metadata_gin ON chat_audits USING gin(metadata);
